Projeto de exemplo: Gerenciador Escolar em Java

Como compilar:

```powershell
mkdir -Force out; javac -d out src/*.java
```

Como executar:

```powershell
java -cp out Main
```

Se estiver usando o botão "Run" do VS Code, abra `src/Main.java` ou use a configuração de execução padrão em `.vscode/launch.json`.
